# 🌌 千星奇域 · 每周经验刷取(回放通关版)

### 🌟 功能介绍

- 🔁 自动重复通关指定奇域关卡。
- 🎥 依靠用户自行录制若干个通关流程，随机抽取一个进行通关。
- 🧺 支持删除关卡存档再挑战（默认禁用），提高刷取效率。
- 📅 自动追踪每周经验获取进度，达到上限后再次执行将跳过。
- 🏞️ 执行完毕后自动返回提瓦特大陆，不干扰其他自动化流程。

## 📋 使用必读

- 若内置通关流程失效，请使用程序的 **[录制回放]** 功能，自行录制 **进入关卡后的通关流程** 。
- 录制完成后，打开回放文件所在目录，将录制的脚本拷贝到本脚本目录下的 `assets/playbacks` 目录中。
- 右键编辑脚本配置，根据你录制的流程调整对应设置（如奇域关卡、通关回放文件池、每次通关获取的经验值等）。
- 建议录制人气不低，不太可能下架，通关流程简单的单人游玩关卡（如石头模拟器、风景打卡类、抽卡模拟器等）。
- 同时可适当地增加录制的通关时长，降低风险。

### 📖 使用方法

- 将本脚本添加至您的配置组中。
- 右键点击脚本自定义配置项。
- 配置好计划通关的关卡信息、通关回放文件等。
- 点击运行按钮。

### 🛠️ 脚本配置

| 配置项                 | 描述                     | 备注                                     | 默认值                        |
| ---------------------- | ------------------------ | ---------------------------------------- | ----------------------------- |
| room                   | 奇域关卡关键词或关卡GUID |                                          | 20134075027                   |
| playbacks              | 通关回放文件池           | 自行录制拷贝到assets/playbacks，逗号分隔 | 通关回放1.json,通关回放2.json |
| expPerAttempt          | 每次通关获取的经验值     | 如果勾选删除关卡存档，请自行增加         | 20                            |
| deleteStageSave        | 删除关卡存档             | 可重复达成成就，获取更多经验值           | false                         |
| deleteStageSaveKeyword | 删除关卡存档关键字       | 关卡存档视图中的[关卡]列                 | 深渊100层                     |
| stageTimeout           | 通关超时                 | 单位：秒，需大于通关回放时长             | 120                           |
| expWeeklyLimit         | 每周可获取的经验值上限   |                                          | 4000                          |
| force                  | 忽略本周经验值已达上限   |                                          | false                         |
| thisAttempts           | 指定通关次数             | 0为自动判断                              | 0                             |
| goToTeyvat             | 完成后返回提瓦特大陆     |                                          | true                          |

### ❗ 注意事项

- 请确保游戏窗口分辨率比例为 `16:9`，否则可能影响脚本正常运行。
- 建议提前设置好快捷键，便于在需要时快速暂停或终止脚本。
